# SpringBoot图书管理系统 java733

## 项目概述

SpringBoot图书管理系统是一个基于Java的项目，采用前后端不分离的模式，实现了图书管理的基本功能，包括图书的增删改查、用户的借还书操作等。

## 技术栈

### 服务端

- **核心框架：** SpringBoot
- **持久层框架：** MybatisPlus
- **数据库：** MySQL

### 前端

- **模板引擎：** Thymeleaf
- **前端库：** bootstrap
- **JavaScript库：** jQuery

## 系统角色

- 管理员：负责图书和用户管理，以及借还书操作等。
- 读者：可以进行借书、还书、挂失、修改个人信息等操作。

## 运行环境

- **数据库：** mysql5.7（兼容mysql8）
- **开发环境：** IDEA
- **构建工具：** Maven
- **JDK版本：** 1.8（推荐）

## 目录结构

```
SpringBoot图书管理系统
├── src
│   ├── main
│   │   ├── java
│   │   ├── resources
│   │   └── webapp
│   └── test
└── pom.xml
```

## 部署步骤

1. 确保MySQL服务已启动，创建对应数据库。
2. 使用Maven构建项目。
3. 部署应用到服务器，如Tomcat。
4. 访问前端页面进行操作。

## 核心亮点

- **响应式设计：** 适应各种设备和屏幕尺寸。
- **高效的后端：** 基于SpringBoot，提供高性能和易维护性。
- **简化开发：** MybatisPlus简化数据库操作，Thymeleaf提供前端模板渲染。
